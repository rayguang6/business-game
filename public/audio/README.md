# Audio Files

This directory contains audio files used in the game.

## Background Music
- `welcome-music.mp3`: Music for the welcome screen.
- `selection-music.mp3`: Music for the industry selection screen.
- `game-music.mp3`: Background music during gameplay.

## Sound Effects

### UI Actions ✅ PARTIALLY IMPLEMENTED
- `button-click.mp3`: Original button click sound
- `Click Button_effect.MP3`: Alternative button click sound
- `Click Button Disabled_effect.MP3`: Button click when disabled/unavailable (not implemented - HTML disabled buttons prevent mouse events)

### Business Actions ✅ IMPLEMENTED
- `Hiring_Lets_Do_This.MP3`: ✅ Enthusiastic hiring confirmation (RANDOM)
- `Hiring_Lets_Go.MP3`: ✅ Excited hiring confirmation (RANDOM)
- `Hiring_Woo_Hoo.MP3`: ✅ Celebratory hiring confirmation (RANDOM)
- `Upgrade_effect.MP3`: ✅ Sound when purchasing upgrades (IMPLEMENTED)

### Service & Customer ✅ IMPLEMENTED
- `service-finished.mp3`: Original service completion sound
- `Job Done_effect.mp3`: ✅ Alternative job completion sound (USED for service completion)
- `Job Lost_effect.MP3`: Sound when jobs are lost or failed

### Game Events ✅ IMPLEMENTED
- `Event Card Pop-up_effect.MP3`: ✅ Sound when event cards appear or show outcomes (USED for both popup and results)
- `Event Card-Good_effect.MP3`: Sound for positive event outcomes (not implemented - using popup sound)
- `Event Card-Bad_effect.MP3`: Sound for negative event outcomes (not implemented - using popup sound)

### Progression ✅ IMPLEMENTED
- `Level Up_effect.MP3`: ✅ Sound when leveling up (USED for level ups)
- `Month Change_effect.MP3`: ✅ Sound when advancing to next month (USED for month changes)
